## Dokumentasi
{: .fs-9 }

Ini adalah dokumentasi pribadi. Tidak ada afiliasi resmi dengan pemilik hak cipta SIMRS Khanza. Dokumentasi ini berisi tentang cara Instalasi, Pemasangan Modul, Update, Panduan Pengguna, Panduan Administrasi dan Panduan Pengembangan. Disajikan dengan bahasa yang mudah dipahami, disertai gambar-gambar dan vidio.

## Getting started

### Dependencies

Just the Docs is built for [Jekyll](https://jekyllrb.com), a static site generator. View the [quick start guide](https://jekyllrb.com/docs/) for more information. Just the Docs requires no special Jekyll plugins and can run on GitHub Pages' standard Jekyll compiler.


### Quick start: Use as a GitHub Pages remote theme

1. Add Just the Docs to your Jekyll site's `_config.yml` as a [remote theme](https://blog.github.com/2017-11-29-use-any-theme-with-github-pages/)

### Local installation
1. Install the Ruby Gem
